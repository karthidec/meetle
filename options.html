🗝️ Google Maps API Key

Each user needs their own free API key. Google gives every account $200 free credit/month — more than enough for personal use.

📋 How to get a free API key

Step 1 — Go to console.cloud.google.com
Step 2 — Create a new project (e.g. "Meetle")
Step 3 — Go to APIs & Services → Library and enable:
   • Maps JavaScript API
   • Distance Matrix API
   • Places API
Step 4 — Go to APIs & Services → Credentials
Step 5 — Click Create Credentials → API key
Step 6 — Copy the key and paste it above
